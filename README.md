# ⚠️ Archive Notice

This repository contains the original source code of **Vjii Robot**, a Telegram management bot developed during the lockdown era.

The project is no longer actively maintained and has been archived for educational, historical, and nostalgic purposes.

This repository remains public so future developers can explore how Telegram bots were built during one of the most active periods of Telegram development.

---

# The Story

During the COVID lockdown years, Telegram became much more than a messaging platform.

For thousands of young developers, it became a place to learn programming, build projects, make friends, and contribute to open source.

One of those communities was **Viji's World**.

The community was founded and led by **MrVjiiKing (Viji)** and became a gathering place for people interested in Telegram bots, Python, automation, and software development.

Vjii Robot was created for that community.

While Viji owned and managed the community, the bot itself was actively developed and maintained by contributors who spent countless hours building features, fixing bugs, helping users, and supporting new developers.

I was one of those developers.

---

# Why This Repository Matters

Back then, many people learned programming through Telegram bots.

Instead of spending my time freelancing, I spent much of it helping others:

* Teaching Telegram bot development

Many developers from those communities eventually built their own projects.

Some became professional developers.

But the lessons, friendships, and memories remain.

This repository is preserved as a reminder of that era.

# Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/Vjii_Robot.git
cd Vjii_Robot
```

Install requirements:

```bash
pip install -r requirements.txt
```

Configure environment variables and bot credentials.

Run the bot:

```bash
python -m VjiiRobot
```
